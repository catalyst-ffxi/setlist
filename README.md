# Set List

Create lists of Bard songs and play them together. This is a work in progress.

## Usage

Use `sl help` to print the options.

Create a list:

```
sl new dummies
sl add "Enchanting Etude"
sl add "Swift Etude"
sl add "Bewitching Etude"
```

Play a list:

```
sl use dummies
sl play
```

Play multiple lists:

```
sl dummies dyna
```
